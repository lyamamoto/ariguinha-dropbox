# K-12: PPM Navigation Guide

**Trigger:** When asked "what to build next", "what's the current state", "where do I find X", or when starting any PPM-related work.

**Content:**

Quick lookup for where to find things:

| Question | File to Read |
|----------|-------------|
| What to build next? | `backlog/prioritization.md` |
| What does a feature require? | `requirements/functional/<domain>/F-NNN-*.md` |
| What's in the current sprint? | `roadmap/sprints/sprint-NNN.md` |
| What's the project state? | `status/project-status.md` |
| Where are test gaps? | `testing/coverage.md` |
| Why was a decision made? | `foundations/adrs/` |
| What does a term mean? | `requirements/domain-model.md` |
| What are the coding rules? | `foundations/coding-standards.md` |
| How do services integrate? | `foundations/integration-contracts.md` |
| What's the high-level timeline? | `roadmap/roadmap.md` |

**Layer flow:** Foundations → Requirements → Backlog → Prioritization → Roadmap/Sprints → Testing → Status (L1 through L7, each feeds the next).
